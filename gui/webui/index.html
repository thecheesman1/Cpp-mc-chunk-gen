Performance Bottleneck Breakdown
Standard JVM Pipeline
  • Dynamic palette linear allocations (O(N) search)
  • Single-threaded per world sector lockups
  • Heavily CPU-bound dynamic Zlib deflate on write
  • 10,000+ GC object allocations per single chunk
McChunkGen Native Pipeline
  • Fixed-palette compilation with direct O(1) layout
  • Multi-threaded lockless spiral walker allocation
  • Post-process offline batch compressor
  • Strict zero heap allocations. Fast stack buffers.
